# Java-Calendar

基于Java的日历
===========

![](https://raw.githubusercontent.com/muyangren907/Java-Calendar/master/screenshots/1.png)

## 说明

- Java编写的日历，仅供参考

## 使用

###	①

- 下载源代码

- 编译

```
javac NewCalendar.java -d ./ -encoding utf-8
```

-	运行

```
java NewCalendar
```

### ②

- 下载jar包运行
